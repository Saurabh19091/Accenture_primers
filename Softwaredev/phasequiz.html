Software Engineering & Logic Quiz (60 Questions)

Quick Revision Notes:

  • Requirements Engineering: Elicitation, Analysis, Specification
  • Functional Requirements: What system should do
  • Non-Functional Requirements: Performance, Usability, Security, Portability
  • Design Phases: HLD (modules), LLD (detailed algorithms), Interface Design, Data Design
  • System Analyst: Creates SRS
  • Design Sequence: Analysis → HLD → LLD → Coding
  • SDLC Models: Waterfall, V-Model, Incremental, RAD, Prototype, Evolutionary Prototyping
  • Scenario-based Questions: Identify suitable SDLC model for a case
  • Logic Puzzles: Carefully read wording to avoid traps.
